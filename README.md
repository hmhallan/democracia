# Democracia
Dapp de votação com ethereum, truffle e angularjs

Exercício da disciplina de mestrado *Tópicos Especiais em Computação: Desenvolvimento Ágil de Sistemas*

(UFSC, 2018/2)

# Deployment

* git clone https://github.com/hmhallan/democracia.git
* cd democracia
* npm install
* npm run dev
